## Lokaverkefni - Inngagnur að máltækni

### Dependencies

```bash
pip install Flask
pip install spacy
python -m spacy download en_core_web_sm
pip install chatterbot
pip install pandas
pip install retry-requests
pip install requests
pip install openmeteo-requests
pip install requests-cache
```

### run
```bash
python3 app.py
```
